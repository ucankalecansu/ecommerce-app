# Ecommerce-App

Bu proje, React Native Expo kullanılarak geliştirilmiş bir mobil uygulamadır.

## Gereksinimler

- Node.js
- Expo CLI
- Yarn veya npm

## Başlangıç

Proje bağımlılıklarını yüklemek için aşağıdaki adımları izleyin:

```bash
# Yarn kullanarak
yarn install

# veya npm kullanarak
npm install


# Yarn kullanarak
yarn start

# veya npm kullanarak
npm start
